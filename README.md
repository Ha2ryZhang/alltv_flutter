# alltv_flutter
本项目基于`flutter`开发，但由于贫穷与mac无缘，所以目前只提供安卓的apk。

### 下载地址
安卓：[Download](https://alltv.lanzous.com/b01bexnha)
#### 如果遇到不能安装的情况，建议卸载重装。
### 后台
后台的话主要要也是自己用Java写的[alltv](https://github.com/ha2ryzhang/alltv),目前开源了大部分吧，只有一部分关于爬虫的暂时还没有开源，主要是太乱了，后面整理整理加进去。
